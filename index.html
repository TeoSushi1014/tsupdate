TSToolkit - Cập Nhật

Trang thông tin cập nhật chính thức cho TSToolkit

Phiên bản mới nhất: 1.1.0

Ngày phát hành: 15/07/2024

Bản cập nhật mới với các tính năng: Cải thiện hiệu suất, sửa lỗi và thêm tính năng mới

Tải về ngay

Các thay đổi trong phiên bản này:

  • Cải thiện tốc độ khởi động
  • Sửa lỗi không thể xóa một số ứng dụng
  • Thêm tính năng tự động kiểm tra cập nhật
  • Cập nhật giao diện người dùng

Hướng dẫn cài đặt:

  1. Tải về file ZIP từ liên kết phía trên
  2. Giải nén file vào thư mục bạn muốn
  3. Chạy file Setup.cmd để cài đặt
  4. Làm theo hướng dẫn trên màn hình
